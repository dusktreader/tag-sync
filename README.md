# tag-sync

![tag-sync icon](docs/source/images/tag-sync-icon.png)

_A CLI tool for syncing git tags with project versions._


## Super-quick Start

Requires: Python 3.13+

Install through pip:

```bash
pip install tag-sync
```

Publish a tag for the current package version:

```bash
tag-sync publish
```
